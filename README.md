Essa é uma API para administrar tarefas

Instruções para utilizar a API:

1 - Clonar o repositório:

git clone https://github.com
cd seu-repositorio-de-tarefas

2- Construir e iniciar a aplicação:

podman-compose up --build -d

3- Para acessar e senha da API:

usuario:adm, sennha:adm
localhost:8000/exibir

4- Para parar a aplicação:

podman-compose down
